# 🛒 Smart Product Recommendation System

A Machine Learning project using **Market Basket Analysis (Apriori Algorithm)** to recommend products based on customer purchase patterns.

---

## 📊 Project Overview

This system analyzes transaction data and finds relationships between products using:

- Apriori Algorithm
- Association Rule Mining
- Confidence & Lift metrics

👉 It answers:
> "If a customer buys X, what are they likely to buy next?"

---

## 🚀 Features

- 🔍 Select any product
- 📊 Get top product recommendations
- 📈 Confidence & Lift based filtering
- 🎨 Premium UI using Streamlit
- 📦 Real-time recommendation system

---

## 🧠 Concepts Used

- Market Basket Analysis
- Apriori Algorithm
- Association Rules
- Support, Confidence, Lift

---

## 📸 Output Example

- Input: **chocolate**
- Output:
  - chocolate → whole milk
  - chocolate → yogurt
  - chocolate → rolls/buns

### 1. Clone the repository

cd smart-product-recommendation

### 2. Install dependencies
pip install -r requirements.txt

streamlit run app.py
